# linux 기초 문법

linux에서 사용 가능한 기초 문법을 정리합니다.



##### 현재 폴더 위치

- pwd (print working directory)

```shell
$pwd
```



##### 폴더 변경

- cd (change directory)

```shell
$cd <이동하고 싶은 폴더 이름>
```



##### 폴더, 파일 출력

- ls(list)

```shell
$ls -a
'-a' 옵션은 숨김 파일까지 출력해준다.
```



##### 파일 생성

- touch
  - 만약 폴더를 삭제하고 싶다면 -r 옵션을 추가해서 삭제 가능

```shell
$touch <생성하고 싶은 폴더 이름>
```



파일 삭제

- ##### rm
  
  - 만약 폴더를 삭제하고 싶다면 -r 옵션을 추가해서 삭제 가능

```shell
$rm <삭제하고 싶은 폴더 이름>
```
